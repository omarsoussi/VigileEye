Login & Authentication Flow

sequenceDiagram
    actor User
    participant Frontend
    participant AuthAPI as Auth API
    participant AuthService
    participant JWTHandler
    participant Database

    User->>Frontend: Enter credentials
    Frontend->>AuthAPI: POST /auth/login
    AuthAPI->>AuthService: login_user(email, password)
    AuthService->>Database: find_user_by_email(email)
    Database-->>AuthService: User
    AuthService->>AuthService: verify_password()
    AuthService->>Database: record_login()
    AuthService->>JWTHandler: create_token(user_id)
    JWTHandler-->>AuthService: JWT Token
    AuthService-->>AuthAPI: AuthToken {access, refresh}
    AuthAPI-->>Frontend: 200 OK {token}
    Frontend-->>User: Login successful

Start Live Stream Flow

sequenceDiagram
    actor User
    participant Frontend
    participant StreamAPI as Stream API
    participant StreamManager
    participant FFMpeg
    participant Camera as Camera/RTSP
    participant FrameEncoder
    participant WebSocket
    participant Browser

    User->>Frontend: Click Watch Camera
    Frontend->>StreamAPI: POST /streams/start
    StreamAPI->>StreamManager: start_stream(camera_id)
    StreamManager->>FFMpeg: Connect to RTSP stream
    FFMpeg->>Camera: Open connection
    Camera-->>FFMpeg: Stream active
    FFMpeg->>FFMpeg: Read frame
    FFMpeg->>FrameEncoder: encode_frame()
    FrameEncoder-->>FFMpeg: Encoded frame
    FFMpeg->>WebSocket: send_frame()
    WebSocket->>Browser: Push frame (WS message)
    Browser->>Frontend: Update video player
    Frontend-->>User: Live feed displayed

    loop Every 33ms
        FFMpeg->>Camera: Read next frame
        FFMpeg->>FrameEncoder: Encode
        FrameEncoder->>WebSocket: Broadcast
        WebSocket->>Browser: Send frame
    end

Invitation & Membership Flow

sequenceDiagram
    actor Inviter

    participant InvService
    participant EmailService

    Invitee->>InviteeApp: Open invitation email
